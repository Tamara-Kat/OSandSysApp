## Завдання 2. Довідка по командах

### Виконані команди

```bash
man ls
help cd
man cat
man man
cp --help
mv --help
```

```
Команда man відкриває сторінку документації для команди.

Команда help використовується для вбудованих команд оболонки, наприклад cd.

Ключ --help показує коротку довідку по команді прямо в терміналі.
```

### Питання 1. Який ключ ls показує приховані файли?

```
Приховані файли показує ключ:

-a

Повна форма:

--all

Приклад використання:

ls -a

Детальний перегляд разом із прихованими файлами:

ls -la

```
### Питання 2. Який ключ у cat нумерує рядки?

```
Рядки нумерує ключ:

-n

Повна форма:

--number

Приклад використання:

cat -n file1.txt
```

### Питання 3. Чим відрізняються man і --help?
```
man відкриває повну документацію команди. У ній зазвичай є опис команди, список ключів, пояснення, додаткова інформація та пов’язані команди.

--help показує коротку довідку прямо в терміналі. Зазвичай там наведені основні ключі та короткі пояснення до них.

man — повна документація команди
--help — коротка швидка довідка
```

## Завдання 3. Міні-сценарій

### Cценарій

```bash
cd Downloads
> scenario.txt
ls
cd ..
pwd
cd Downloads
cat scenario.txt
man ls
```

### Пояснення сценарію

```
cd Downloads — перехід у каталог Downloads.
> scenario.txt — створення порожнього файлу scenario.txt.
ls — перегляд списку файлів.
cd .. — перехід на один рівень вище.
pwd — перегляд поточного розташування.
cd Downloads — повернення назад у каталог Downloads.
cat scenario.txt — перегляд вмісту файлу.
man ls — перегляд документації команди ls.
```
