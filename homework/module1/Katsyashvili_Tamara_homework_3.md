# Homework_3 Процеси та ресурси Linux

## Завдання 1. Огляд активних процесів

### Виконані команди

```bash
ps aux
top
echo $$

```

### Результат
```
За допомогою команди:

ps aux

було виведено список усіх активних процесів у системі.

Для моніторингу процесів у режимі реального часу використано команду:

top

У вікні top було знайдено процес, який використовує найбільше оперативної пам'яті. На момент виконання завдання ним був процес Xorg.

PID поточної оболонки було визначено командою:

echo $$

Результат:

2047
```

## Завдання 2. Робота у фоні та керування процесами
### Виконані команди

```bash
sleep 1000 &

jobs

fg

Для зупинки процесу використано комбінацію клавіш:

Ctrl + Z

Після цього:

jobs -l

kill -9 PID

Запуск процесу за допомогою nohup:

nohup sleep 1000 &

Результат:

Процес було успішно запущено у фоновому режимі.

Вивід команди:

[2] 2346
nohup: ignoring input and appending output to 'nohup.out'
```

## Завдання 3. Пріоритети та обмеження

### Виконані команди

```bash
nice -n 10 sleep 1000 &

Перевірка процесу:

ps aux | grep sleep

Зміна пріоритету вже запущеного процесу:

renice 15 -p 2346

Перегляд обмежень ресурсів користувача:

ulimit -a
Результат

Змінено пріоритет процесу:

2346 (process ID) old priority 10, new priority 15

Отримано список поточних обмежень ресурсів користувача за допомогою команди:

ulimit -a
```

## Завдання 4. Моніторинг ресурсів

### Виконані команди

```bash
df -h

free -h
Результат

Отримано інформацію про:

використання дискового простору;
обсяг вільної та зайнятої оперативної пам’яті.

Команда:

df -h

виводить інформацію про файлові системи та використання дискового простору.

Команда:

free -h

виводить інформацію про використання оперативної пам’яті у зручному для читання форматі (МБ/ГБ).

```
